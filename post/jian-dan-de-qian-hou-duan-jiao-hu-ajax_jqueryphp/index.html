简单的前后端交互--Ajax_ jQuery，PHP

2020-04-15 / 2 min read

一，利用Ajax_ jQuery给后端提供数据

前端代码如下：

<div class="captcha">
	<button>测试</button>
		<script>
			$("button").click(function(){
				$.post("./api/captcha.php", //URL
					{
						name:"伊兹和刃唯阿都是我老婆",
						url:"https://www.dqdxz.com"
					},
				function(data,status){
				alert("数据: \n" + data + "\n状态: " + status);
						});
					});
				</script>
</div>

（1）利用$("button").click( ），获取button的节点，并且绑定click点击事件；

（2）利用$.post() 方法通过 HTTP POST 请求向服务器提交数据；

（3）以及最后的回调函数，确认成功

二，后端接收数据

后端代码如下：

<?php
$name = $_POST['name'];
$url = $_POST['url'];
print_r("我接受到了".$name.$url);
echo "success";
?>

（1）利用 $name = $_POST['name']; $url = $_POST['url']; 获取前端数据

（2）print_r(）函数输出数据 data，以及通过status获得状态，success
